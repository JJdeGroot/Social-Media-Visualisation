Requires a file named "config.js" that contains the following variables:

var twitter_details = {
    consumer_key:         '',
    consumer_secret:      '',
    access_token:         '',
    access_token_secret:  ''
};

var instagram_details = { 
    client_id: '',
    client_secret: '' 
};

var tumblr_details = {
    consumer_key: '',
    consumer_secret: '',
    token: '',
    token_secret: ''
};

Steps
1. Put in your keys
2. npm install
3. app js
4. navigate to http://127.0.0.1:3000/
